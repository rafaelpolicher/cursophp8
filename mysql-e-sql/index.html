mysql

mysql e mariadb sao extremamente parecidos
bases de dados relacional, dados sao armazenados em tabelas, as tabelas podem ter ralaçoes entre si atraves de chaves, a comunicaçao é feita atravez de SQL

SQL structured query language
linguagem usada para construir queries, que sao instruçoes para comunicar com a base de dados
com SQL podemos pesquisar, inserir, atualizar e eliminar dados, bem como altera a estrutura da nossa base de dados
