bases de dados relacional, dados sao armazenados em tabelas, as tabelas podem ter ralaçoes entre si atraves de chaves, a comunicaçao é feita atravez de SQL

SQL structured query language
linguagem usada para construir queries, que sao instruçoes para comunicar com a base de dados
com SQL podemos pesquisar, inserir, atualizar e eliminar dados, bem como altera a estrutura da nossa base de dados

relaçoes

primary key valor unico usado para relacionar dados entre tabelas, so pode exestir uma PK por tabela
foreign key valor que da suporte a chave primaria na execuçao de queries

relaçao de um para muitos uma tabela 'pessoas' e uma tabela 'emails' uma pessoa pode ter varios emais, mas um email nao pode ter varias pessoas

relaçao de muitos para muitos imagine uma tabela 'encomendas' em que cada encomenda tem um id, e uma 'produtos' que cada produto tem um id, e uma tabela que liga os dois id... a encomenda id 1, tem produto id 8
